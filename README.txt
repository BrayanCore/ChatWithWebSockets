COMO EJECUTAR LA APLICACIÓN:

1. Abrir terminal dentro de la carpeta 'Stratplus'
2. Entrar a la carpeta 'server' -> cd server
3. Encender el servidor -> npm run dev
4. Abrir otra terminal dentro de la carpeta 'Startplus'
5. Entrar a la carpeta 'client' desde la terminal -> cd client
6. Ejecutar el comando 'ng serve' -> ng serve --o, el cual abrirá 
una pestaña en el navegador con la interfaz del software una vez 
que se termine de procesar el software
7. Cada pestaña equivale a un usuario en el chat
8. Si se quiere entrar como nuevo usuario al chat, 
entrar mediante 'http://localhost:4200/'
